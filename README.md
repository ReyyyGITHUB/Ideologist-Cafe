# Ideologist Cafe Website

Situs landing untuk Ideologist Cafe: info lokasi & jam buka, menu kurasi dengan unduhan PDF, galeri, serta reservasi meja yang terhubung ke WhatsApp.

## Tech Stack
- React 19 + Vite
- Tailwind CSS 4 (utility + token kustom)
- React Router
- lucide-react (ikon)

## Run & Build
- Install dependencies: `npm install`
- Dev server: `npm run dev`
- Build produksi: `npm run build`
- Preview build: `npm run preview`

## Env
- Tidak ada env khusus yang wajib. Pastikan aset gambar/PDF tersedia di `/assets`.
